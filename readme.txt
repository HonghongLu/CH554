Software brief:

CH554 evaluation board description and reference application routines, including USB devices, USB-Host, read and write U disk files, Touch-Key, Type-C, ADC, timer, PWM, serial port, SPI and so on.


Scope of application: CH554T, CH554G, CH554E, CH553G
Download and install the software:
WCHISPTOOL (ISP download does not need to install other drivers separately)


Download method:
DOWN and 3V3 use tweezers such as short connection, insert USB, disconnect short connection immediately (because DOWN and USB are reused), and WCHISPTOOL software can identify USB devices at this time.
